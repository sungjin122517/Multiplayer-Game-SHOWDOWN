Game Description

This game is a 2 player game. Shoot the opponent when you see 'SHOWDOWN'!

Each player has 3 HPs. pull the trigger FASTER than the opponent before it shoots you.

Warning: You will get stunned if you pull the trigger before 'SHOWDOWN!'

Good luck!

Controls

Press the spacebar to pull the trigger.
